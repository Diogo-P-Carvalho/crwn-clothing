<!-- GETTING STARTED -->

## Getting Started

1. Clone this repo, and the API
    ```sh
    git clone https://github.com/Diogo-P-Carvalho/crwn-clothing.git
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Run npm start
    ```sh
    npm start
    ```
